# pi-extmgr

Enhanced UI/UX extension for managing Pi extensions and discovering community packages.

## Installation

```bash
pi install npm:pi-extmgr
```

## Usage

```
/extensions              # Open interactive manager
/extensions list         # List local extensions
/extensions installed    # List installed packages
/extensions remote       # Browse community packages
/extensions search <q>   # Search npm for packages
/extensions install <s>  # Install package
/extensions remove <s>   # Remove package
```

## Keyboard Shortcuts

- `↑↓` Navigate
- `Space/Enter` Toggle enabled/disabled
- `S` Save changes
- `I` View installed packages
- `R` Browse remote packages
- `?` Help
- `Esc` Cancel
